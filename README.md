# Eaglercraft-Links-and-Servers
An Eaglercraft links and servers directory, hostable with GitHub Pages.

## Pages structure
- `/index.html` - root directory page
- `/versions/1.5.2/` - Eaglercraft 1.5.2 links directory
- `/versions/1.8.8/` - EaglercraftX 1.8.8 links directory
- `/servers/` - public server directory page

Enable GitHub Pages on this repository and use the root branch to host the directory.
